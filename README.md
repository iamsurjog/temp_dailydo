# DailyDo
## Installation
To download the repo run
```bash
 git clone https://github.com/iamsurjog/temp_dailydo.git
 cd temp_dailydo
 py -m venv ./.venv
 pip install -r requirements.txt
```

## Running it
### Windows
```bash
.\.venv\Scripts\activate
flask --app app run 
```

### Linux
```bash
source .venv/bin/activate
flask --app app run 
```
